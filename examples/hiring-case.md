# Hiring Case Example

## JD

Senior AI/RAG Engineer responsible for production RAG architecture, retrieval quality, evaluation, deployment, and monitoring.

## Candidate Notes

Candidate claims:

- Built a production RAG system for customer support.
- Improved retrieval accuracy through hybrid search and reranking.
- Designed an evaluation pipeline for answer faithfulness.
- Deployed monitoring for hallucination and retrieval failures.
- Integrated a vector database with permission-aware retrieval.

## Verification Focus

- Ask for one production failure and the root-cause process.
- Ask for evaluation metrics, baselines, and ablation results.
- Ask how retrieval errors are separated from generation errors.
- Ask what the candidate personally owned versus what the team owned.
- Ask how the system behaves when documents are stale, duplicated, or permissioned incorrectly.
