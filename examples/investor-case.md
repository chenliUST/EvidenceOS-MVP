# Investor Case Example

## Deck Notes

Acme AI claims it helps enterprise teams deploy agentic RAG over internal knowledge bases.

Claims:

- 12 paying customers.
- 30% answer quality improvement over baseline RAG.
- Proprietary evaluation pipeline.
- Security-aware retrieval and permission filtering.
- Median implementation cycle is two weeks.

## Founder Call Notes

Founder says the main technical edge is orchestration around retrieval, reranking, tool use, and evaluation.

The current team includes two infrastructure engineers and one GTM founder. The founder says they are focused on regulated enterprise buyers but has not yet provided a security review artifact.

## Key Verification Questions

- Which metric defines the 30% answer quality improvement?
- Which customers are active in production today?
- What happens when documents are stale or permissions are wrong?
- What is proprietary in the evaluation pipeline?
- Which buyer signs the contract and which user logs in weekly?
